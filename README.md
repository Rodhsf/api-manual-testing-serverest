# API Testing - Serverest

Projeto de testes de API utilizando Postman para validar endpoints da API Serverest.

---

##  Objetivo

Validar funcionalidades da API através de testes positivos e negativos.

---

##  Ferramentas

- Postman
- JSON

---

##  Funcionalidades testadas

### Login
- Login com dados válidos
- Login com dados inválidos

### Usuários
- Criar usuário
- Criar usuário sem dados
- Listar usuários

### Produtos
- Listar produtos
- Cadastrar produto
- Deletar produto

---

##  Estrutura

api-testing-serverest/
├── postman/
├── evidences/
├── docs/
└── README.md


---

## ▶️ Como executar

1. Importar a collection no Postman  
2. Executar os requests  
3. Validar as respostas  

---

## 📸 Evidências

Os prints dos testes estão na pasta `evidences/`.
